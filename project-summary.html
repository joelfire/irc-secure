Project Information

Field Value
Name irc-secure
Description This is a simple encryption mechanism for IRC. It differs from SSL in that it secures data end to end, not just client to server. Currently it just uses a shared key. All users must run this client, and have the same key. Any other user will just see the encrypted data.
